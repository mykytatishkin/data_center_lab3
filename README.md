# Lab 03 - Power Redundancy in Data Centers

Understanding power redundancy models (N, N+1, 2N+1) and simulating power failure scenarios.

## Deliverables

| File | Description |
|------|-------------|
| `lab03-power-redundancy.html` | **Main lab report** – Open in a browser to view all diagrams, failure scenarios, and discussion answers |
| `drawio/01-N-model.drawio` | N model diagram (draw.io format) |
| `drawio/02-N+1-model.drawio` | N+1 model diagram (draw.io format) |
| `drawio/03-2N+1-model.drawio` | 2N+1 model diagram (draw.io format) |

## How to Use

1. **View the lab report**: Open `lab03-power-redundancy.html` in any web browser.
2. **Edit diagrams in draw.io**: Open the `.drawio` files in [draw.io](https://app.diagrams.net/) or VS Code with the Draw.io extension.

## Lab Summary

- **Task 1**: Power redundancy models (N, N+1, 2N+1) with components: Utility Feed → Generator → UPS → PDU → Server
- **Task 2**: Failure scenarios (A: PDU failure, B: UPS failure, C: path failure) with impact analysis
- **Discussion**: Answers to the four discussion questions
